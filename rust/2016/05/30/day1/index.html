Rust - Day 1

William and his friend is going to rewrite their game in Rust. I am going to join them. So I start to learn Rust.

What is Rust?

Rust is target to be a new generation of system programming language

Cargo

It is an package manager, just like npm for NodeJS.

Use cargo new <application name> --bin to create a repo for an application. If you want to create a library, omit the --bin option. After setting up a package, we can use cargo build to compile and cargo run to run rust code easily.

Crates

It means libraries and packages in Rust
