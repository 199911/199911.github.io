7 Rules for Creating Gorgeous UI

  • Light comes from the sky
  • Black and white first
    • Designing in grayscale before adding color simplifies the most complex element of visual design– and forces you to focus on spacing and laying out elements.
  • Double your whitespace
  • Learn the methods of overlaying text on images
    • Tips
      • Image should be dark
      • Text has to be white
      • Test it at every screen/window size
    • Methods
      • Overlay the whole image
      • Text-in-a-box
      • Blur the image
      • Floor fade
        • subtly fades towards black at the bottom, and then there’s white text written over it
  • Make text pop and un-pop
    • Use style to increase/decrease visibility of the text
  • Only use good fonts
  • Steal like an artist
    • Listed some website for learning from other’s design work
