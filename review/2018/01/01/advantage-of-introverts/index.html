內向者的優勢：安靜的人如何展現你的存在，並讓別人聽你的

  • 內向 vs 外向
    • 獲取能量差異
      • 內向者需要抽離與安靜
      • 外向者需要從活動與他人的接觸
    • 內向同外向係一個連續譜系
      • 長時時超出自己嘅舒適區對健康有影響
    • 係內向同外向之間遊移係正常
      • 人會受到文化，情境，扮演嘅角色，年紀，心情而變動
    • 內向，害羞，高度敏感係三種冇直接關聯的特質
    • 內向唔係一個缺點
    • 對自己嘅需求越清楚，就能越輕鬆面對自己同其他人相處
  • 內向者的長處
    • 謹慎，實在，專心，傾聽，平靜，擅長分析，獨立，穩定，書寫（取代談話），為人著想的能力
  • 內向者的障礙
    • 恐懼，瑣碎，過度刺激，消極，逃避，過於尚智，自我否定，僵固，避免接觸，畏懼衝突
  • 與外向者溝通策略
    • 講重點
      • 先用紙筆整理要講嘅重點
    • 音量充足，要有語調
    • 可以減慢對話節奏
    • 要有充足時間考慮
    • 表達自己嘅情感
  • 家庭生活策略
    • 安排抽離區作休息用途
    • 處理噪音
      • 定立室內響度
      • 爭吵緊張時安排暫定時間
      • 耳機
    • 保母
      • 幫手照顧細路
    • 精神食糧
    • 運動
  • 團隊工作
    • 長時間不受干擾
      • 提早開始，或比人留更晚，爭取無人時間
      • 會議後保留時間用作溝通
      • 與同協調一個可以安靜工作嘅時間
      • 分配好時間，每隔一段時間休息一下，交換意見
      • 約不方便談話，可以另約時間
    • 間中平靜一下
      • 將工作當作舞台，要有某種程度嘅表現，都需要揾機會休息
      • Prioritize social event
    • 有目標地進行溝通
      • 列出完成嘅工作
      • 表現出有關心別人，講對人地重要嘅野，要有細節
      • 總結重點同促成決定
      • 展現團隊成果
      • 有新同事或任務，要說明responsibility同expectation
      • 有困難時，要提出黎同他人磋商
      • Set up milestone，同同事慶祝，增進感情
  • 管理策略
    • 建立自信
      • Keep track 自己完成嘅工作
    • 認真關注他人
      • 同理心
      • 不帶成見傾聽
      • 虛心接受他人意見
    • 培養綜觀事物同掌握全局嘅能力
    • 訓練對話同面對衝突
      • 四處走動同人地溝通
      • 要關注商討衝突
    • 彰顯成績
      • 每半年檢視自己完成嘅工作
      • 在工作領域內外建立專業人脈
      • 表明興趣所在，朝自己擅長嘅領域進發
      • 勇於任事
      • 下放權責
    • 善用溝通管道
      • 電話
        • 適合用嘅情況
          • 可以簡短，直接，迅速地說明
          • 易做成誤解，但不能面對面相談
          • 交涉敏感事項
        • Tips
          • 做筆記
          • 留言要簡單留低資訊，名，來電緣由，電話號碼，其他要求（如有），表達問候
          • 可以同人地confrim通話所需時間
      • Email
        • 適合用嘅情況
          • 要有record
          • 攰嘅時候
          • 會面前嘅breifing
          • 當對方係陌生人
      • 出差
        • 尋找抽離空間
          • 由時間回房休息
          • 坐頭等
          • 用盥洗室作抽離空間
          • 耳塞
        • 梅花間竹安排接觸同獨處時間
        • 調節旅行對話
  • 如何展現自己的存在、獲得他人的傾聽？
    • 85%管理職位係透過關係任命，係熟人或推薦
    • 如何建立適合你並可發揮長處嘅人脈
      • 設定人脈活動所追求嘅目標
      • 定義自己嘅資源
        • 時間，金錢
        • 介紹朋友互相認識
        • 請朋友介紹
        • 持之以恆
          • 建立人脈要長時間嘅effort
    • 後續聯系tips
      • 以書信，卡片聯系，較有誠意
      • 寫一D有運用價值的內容
        • 網址，文章同書等等
      • 相識2至4日聯系，加深impression
    • 閒聊stragegy
      • 重質不重量
        • 集中同3至4人單獨聊天
      • 閒聊前先揾抽離方法同空間
      • 尋找可能嘅對話者
        • 可以約先前認識嘅人一齊去
      • 先計劃目標
    • 保存能量tips
      • 不要讓自己處於壓力之下
      • Don’t multitask
      • 避開噪音
    • 增加參與度
      • 幫手organize event
        • 了解如何運作
      • 早點到場
      • 排隊時同前後嘅人傾計
      • 善用站桌
    • 利用Web2.0
      • 在一至兩個social platform 保持active
      • 不要貿然將網絡接觸當成友誼或職場上嘅關係
      • Set up固定時間maintain social platform
  • 談判交涉
    • 事前準備
      1. 確定觀點與立場
        • 你必須提供些什麼?
        • 你想透過談判交涉達成什麼目標?
        • 你的交涉對象如何看待這個目標?
        • 你需要哪些與對方以及談判目標有關嘅資訊?
      2. 區別重要的與可妥協的部份
        • 確認重要議題
        • 思考討論順序
        • 你希望最好嘅情況下個別議題分別達成什麼?
        • 點樣強差人意嘅結果係你仲可以接受
      3. 確保目標明確一致
        • 若以團體代表出席，必先向團隊每人釐清目標
    • Tips
      • 要保持靈活彈性
      • 確保交談框架
        • 堅持每個人都暢所欲言
        • 若被對方打斷，可和顏悅色地說：「請讓我很快把話說完」
        • 若有人離題，可以和善地拉回正題
        • 盡可能在措詞上迎合交涉對象
        • 保持身體面向對方，友善地注視對方雙眼
        • 避免交叉手同翹腳
        • 面對挑釁或時間緊迫，要保持鎮靜，可以平靜地深呼吸
      • 聰明傾聽的3個重要問題
        • 聽到咩需求？
        • 聽到咩感受？
        • 有咩拓展對話嘅可能？
      • 留意有助對話進行嘅資訊
      • 不要反覆不變重申自己立場
        • 可直接用語言引導到你在意嘅事
      • 談判唔係講道理，有理據唔一定勝出
        • 要留意對方嘅情感
      • 在交涉中jot note，以掌握全盤局勢，保持靈活
      • 面對對手施壓
        • 結果係要雙方一同決定，冇人可以強迫你接受
        • 要保持自己步調同策略
        • 可先表達對對方重要立場嘅理解
        • 要從容不逼地注視對方，專注但不呆滯
        • 可以用沉默去爭取思考時間，冇人可以逼你迅速表態
        • 可以爭取暫停
  • 演講
    • 成功演說三大準則
      • 名實相符地表現自己
      • 傳達了清楚明確嘅訊息
      • 配合觀眾及其需求
    • 組成部份
      • 開場
        • 喚起對主題嘅興趣
        • 確保主旨的址向
      • 主要內容
      • 結尾
        • 清楚明白地傳達主旨同訴求
        • Call to action
        • 尋求正面結語
    • 對抗恐懼
      • practise
      • 心理策略
        • 對自己講已做好準備
        • 對自己講值得為內容冒險
        • 深呼吸
        • 擺出自信嘅姿勢
    • 平衡重點同細節
      • 演講照架構進行
      • 處理細節時，要思考對主旨有冇幫助
      • 多出來嘅資料可當作補充資料，有聽眾想知道細節先講，不需要演講中仔細說明
    • 要同觀眾交流，唔好照稿讀
  • 參加會議與討論
    • 會議守則
      • 避免發言過於瑣碎，要令其他人聽得入耳
      • 聲量要足，速度要適中
      • 要與其他人有目光接觸，特別係決策者
        • 可以避免他人偷橋
      • 爭取發言，最少同地位相近嘅同事一樣長
      • 盡量說服層級較高嘅人
      • 避免伸懶腰
      • 處理違規Tips
        • 被人打斷
          • 若係同級或下級可堅持說完
          • 若係上級，最好先讓對方先講
        • 他人解釋你嘅意見
          • 地位競爭，表現對你的發言有解釋權
          • 不太同意，有時候需要補充資料
        • 意見被拖延
          • 意見被暫緩進行而動力全失
        • 有人反對先前達成的協議
          • 可以私下先談一談
          • 若有正式協議，應先直接表明
        • 被人不公平攻擊
          • 針對事實回應
      • 重要嘅決議多數在會議之前已經喬好
        • 應為一個盡可能對大家有利嘅決議鋪路
      • 決議後要follow up
        • 執行都好重要
        • 要留有會議記錄
    • 主持議事
      • checklist
        • 時間 (starting time & duration)
        • 地點
        • 會議主旨及目標
        • 哪些議題要排入議程
        • 討論順序
        • 每項議題要花商議時間
        • 若時間有限，哪些議題可以放棄
        • 與會者清單
        • 誰負責主導哪個議程
        • 要為誰提供書面資料
        • 誰紀錄會議
        • 需要哪些器材?
        • 誰在何時邀集與會者?
        • 誰確保會議資料完整性
        • 誰負責會場準備
      • 處理會議中麻煩的情況
        • 冷場
          • 做總結
          • 整理未解決問題
          • 引導他人發言
        • 意見分歧
          • 以中立角度解釋2個立場嘅差異
          • 可暫定會議若有人情緒太激烈
        • 被與會者批評
          • 若真有其事，要體諒他人，解釋說明
        • 挑釁或攻擊
          • 保持冷靜
          • 以事論事
      • 處理會議中麻煩的與會者
        • 話多者
          • 幫對方或請對方總結
        • 強勢者
          • 先同佢地溝通
          • 鼓勵其他人發現
        • 具攻擊性者
          • 平心靜氣處理
          • 降低回應對方嘅音量
        • 易怒者
          • 平心靜氣處理
          • 問佢有何不滿
        • 悲觀者
          • 評估提出嘅疑慮是否合理
          • 討論解決方法
          • 提出尋找專業人士提供意見
        • 打斷其他人發言者
          • 暫停發言，將目光放係交談者身上
          • 先讓發言者說完，再安排打斷發言者發言
